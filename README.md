# Bank-Analytics-Dashboard
Bank Analytics Dashboard using Excel, Tableau, Power BI

## ðŸ“Œ Project Overview

The **Bank Analytics Dashboard** project is designed to analyze banking and loan-related data to gain insights into loan performance, repayment behavior, customer demographics, and regional trends.

The same dataset and business problem are visualized using **Excel, Power BI, and Tableau** to demonstrate the ability to work across multiple analytics tools and present insights effectively to stakeholders.

This project helps banks and financial institutions to:

* Monitor loan disbursement and recovery
* Identify high-performing regions
* Analyze customer segments
* Track interest rate and payment trends
* Support data-driven decision-making

---

## ðŸŽ¯ Business Objectives

* Analyze **total loan amount, funded amount, and payments**
* Track **loan status** (Active, Fully Paid, Net-Off, etc.)
* Understand **regional and demographic distribution**
* Compare **funded amount vs total payment over time**
* Evaluate **interest rate trends**
* Assess customer segments such as **caste, gender, and product type**
